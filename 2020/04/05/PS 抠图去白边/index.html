PS 抠图去白边

Posted on 2020-04-05 Views:

1. 选区缩像素

  1. 按住Ctrl键，点击人物层缩略图，为人物建立选区；

  2. 点击“选择”→“修改”→“收缩”，将收缩量调整为“2像素”，将选区收缩2像素

  3. 按住Ctrl+Shift+I 反选选区，按下Delete键删除，就能完美地去除白边了。

  4. 按Ctrl+D取消选区。

2. 滤镜处理法

  1. 按住Ctrl键，点击人物层缩略图，为人物建立选区；

  2. 点击“图层”》“图层蒙版”》“显示选区”，为人物层添加图层蒙版；


3. 点击“滤镜”→“其他”→“最小值”，根据画面效果调整“最小值”，将保留值修改为“圆度”，最后直到白边消失即可；
4.

3. 利用“选择并遮住”

  1. 按住Ctrl键，点击人物层缩略图，为人物建立选区；

  2. 点击“选择”→“选择并遮住（调整边缘）”；

  3. 首先勾选“智能半径”、“净化颜色”，然后分别调整“半径值”、“移动边缘”、“羽化值”（净化颜色数量直接保持默认即可）三个选项，直到白边消失为止，同样也能很好地消除白边；
